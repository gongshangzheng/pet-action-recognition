# 宠物动作识别

聚焦宠物（猫/狗）动作识别方向，覆盖文献调研、数据集构建、基线复现到细粒度方法探索的全流程。

## 项目背景

核心挑战是**粗粒度与细粒度动作之间的性能鸿沟**：主流方法在粗粒度动作（行走、站立、奔跑）上已达 88%+ 准确率，但细粒度动作（如「反刍-躺卧」vs「反刍-站立」）仅 12.7%–29.6%。

## 技术路线

- 2D CNN → 3D CNN → Transformer → Skeleton → 多模态 → 视频基础模型
- 级联架构：姿态提取 → 粗粒度分类 → 细粒度识别
- Stitching-Retargeting 范式：缝合个体表征 → 重定向动作识别能力
- 部件级时序建模（PMTNet 路径，猫行为专用）

## 协作方式

大部分操作可通过 AI 协作完成，配套 Claude Code skills 提供操作指南。
