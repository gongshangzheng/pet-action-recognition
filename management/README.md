# Management — 项目管理体系

ProjFlow 的多人协作文档体系，提供结构化的项目管理机制。

## 目录结构

```
management/
├── team/            # 团队成员档案
│   ├── README.md    # 团队总览
│   └── {姓名}.md    # 成员档案（按需创建）
├── daily/           # 日报
│   └── template.md  # 日报模板
├── weekly/          # 周报
│   └── template.md  # 周报模板
├── monthly/         # 月报
│   └── template.md  # 月报模板
└── docs/            # 项目管理文档
    ├── tasks.md     # 任务规划与跟踪
    ├── milestones.md # 里程碑
    └── meetings/    # 会议纪要
        └── template.md
```

## 使用规范

### 日报

- 路径格式：`daily/YYYY/MM/DD-姓名.md`
- 示例：`daily/2026/07/09-张三.md`
- 每日下班前提交，记录当天工作进展、遇到的问题与明日计划

### 周报

- 路径格式：`weekly/YYYY/WXX-姓名.md`
- 示例：`weekly/2026/W28-张三.md`
- 每周五提交，汇总本周工作、实验结果与下周计划

### 月报

- 路径格式：`monthly/YYYY/MM-姓名.md`
- 示例：`monthly/2026/07-张三.md`
- 每月最后一个工作日提交，阶段性总结与下月规划

### 团队成员

- 每位成员维护一份档案（`team/姓名.md`），记录基本信息、研究方向与技能特长
- 新成员入职时创建档案
